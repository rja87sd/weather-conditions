Week 6 Day 2 Assignment | Classes and Extensions

Instructions:

Part 1: Create the Base Weather Class

  1. Create a new repository named weather-conditions.
  2. Add an index.html file and a script.js file.
  3. In script.js, define a class called Weather.
  4. The Weather class should have properties for temperature, precipitation, and description.
  5. Include a method in Weather to display general weather conditions.

Part 2: Extend the Weather Class

  1. Define a class DayWeather that extends the Weather class.
  2. Define a class DayWeather that extends the Weather class.
  3. Override the display method in DayWeather to include the additional information specific to the day.

Part 3: Instantiate and Test Classes

  1. Instantiate the Weather class to represent general conditions.
  2. Instantiate the DayWeather class for a specific day's conditions.
  3. Call the display methods on both instances and ensure they output relevant information to the console or the web page.

Part 4

  1. In the README.md of your repository, document the purpose of each class and how they are related.
  2. Include examples of the output from your display methods.
  3. Push your changes to GitHub.
  4. Deploy your project using GitHub Pages or a similar platform.
  5. Submit the URL link to your GitHub repository and the live site as your assignment submission.
